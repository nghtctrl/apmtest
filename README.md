# Getting Started with Local Development

## Prerequisites

You will need access to the project `.env` file, which includes the `DATABASE_URL` for the shared dummy database. Please contact us to obtain the `.env` file.

## One-Time Setup

1. Clone this repository.

2. Place the provided `.env` file in the root directory of the cloned repository.

3. Install dependencies:

   ```bash
   npm install
   ```

## Running Locally

To run the app locally at any time:

```bash
npx netlify dev
```
